Audio Transcription Analysis UI

This web-app enables a structured side-by-side comparison of ASR transcriptions. The core functionality is as follows:

üìÑ Transcribe File

Upload an existing WAV audio file and get its transcription. Features include:

  • Support for WAV audio files
  • Side-by-side view of formatted text and raw JSON
  • Download transcription in JSON format with word-level timestamps

üéôÔ∏è Record & Transcribe

Record audio directly from your microphone and get it transcribed. Features include:

  • Direct microphone recording
  • Live recording status indicator
  • Download options for both audio and transcript
  • Instant playback of recorded audio

üîç Compare & Annotate

Compare and evaluate two different transcriptions of the same audio file. Features include:

  • Automatic statistics:
    • Word count comparison
    • Word Error Rate (WER) calculation
  • Interactive playback:
    • Synchronized word highlighting
    • Click-to-play from any word
    • Side-by-side transcript viewing
  • Quality assessment:
    • Structured rating system
    • Evaluation of accuracy, punctuation, and timing
    • Results saved with comprehensive metadata
